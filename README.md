# ElasticModel

ElasticModel is a lightweight ruby gem to integrate Elasticsearch to your Rails
models.

## Installation

Add this line to your application's Gemfile:

    gem 'elastic_model'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install elastic_model
